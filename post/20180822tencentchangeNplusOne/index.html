【随笔】改变鹅厂N+1人文案整理输出

发表于 2018-08-22 | 分类于 随笔 |

本文是给公司健身房的推广文案，刚好算是简短总结这两年的健身经历。

原标题：《 昔日龌龊不足夸，今朝放荡思无涯 》

Everybody，我叫yofer，从16年3月开始健身至今，2.5年了。

上面的对比图呢，只是健身前后的对比图，冲击力不够，下面再给一张最胖时期的生活照和现在的对比：

还是那句老话，“胖子都是潜力股！”

减肥&健身历程

  1. 胖子：初中时候体重达到峰值256**斤，体重增长太快导致走路脚后跟都会疼痛。（类似孕妇的感觉）
  2. 节食：高中忍受不了自己的肥胖，开始减肥。但刚开始减肥主要靠节食，瘦的很快，但遗留下来的生长纹，现在都还留有印记。
  3. 改变：大四的时候开始尝试健身，第一张对比图就是开始健身时候的样子。小腹翩翩体型弯曲。
  4. 锻炼：刚开始不太会系统性训练，而且身体对于运动的适应性很差，大概到16年11月，慢慢适应了每周两三次的健身房运动强度，但依旧是每次去挨个器械来一遍，形体依旧没有变化。
  5. 健身：16年11月开始在 youtube 上找国外的健身指导视频，从健身的强度变化，动作要求，系统性原则，饮食规范等，全面了解高效率、并能改变视觉效果的健身方法。不断地学习，在自己的身上做尝试和调整，才逐渐有了现在穿衣脱衣都能秀的身材。

踩过的坑&建议

  1. 节食：减肥千万不要单纯靠控制饮食，就算瘦下来，肩膀和腰上都是很明显的生长纹，几乎是很难消除的。而配以运动量的缓慢而持续性的减肥，会让你的皮肤自然收缩，效果就好很多。
  2. 锻炼：不要一开始就想达到欧美级别的运动量，就想变成肌肉男，就想穿衣显瘦脱衣有肉。一定要循序渐进，我现在带着很多同事去健身，刚开始我给他们的预期是改善平日工作状态的疲惫，而并不是马上就能变成我这样有线条的身材，甚至都不提这点。《柳叶刀》杂志发表的最新研究也表示，每周运动3~5次，每次45分钟，对于提升精神状态来说，效果最好。就算每周2次，也对自己的身心都会有益的，经过一两个月，身体适应了运动强度，然后再逐渐增加频率，增加每次的运动量。
  3. 健身：是否想要令人羡慕的身材，这是自己的收获，不要被过高的目标，而丧失了运动有益身心这个最本质收获。
  4. 动力：身体是个可以视觉化的变量，多拍照，阶段性的对比，就会形成正向的循环。
  5. 教练：公司这两年多，教练们基本都认识了，都是非常nice的，日常答疑，指导过动作，推荐过徒手训练方法，组织健身工厂活动。平日更是有非常多的健身小课程。这样优质的资源，这样nice的教练们，大家都不要放过了。

收获

  1. 身材：从此穿衣效果超过模特，一般模特身材撑不起来衣服，垮垮的不好看，哈哈哈。
  2. 精神状态：刚来公司的5个月，虽然断断续续去健身房运动，但没有养成习惯，长期瘫坐在工位上，形体越来越难看，体重也不断升高，工作效率也越发低下，甚至常常对自己的状态感到烦躁。健身后平日的精神状态有极致的提升，16年和现在相比，身体素质的提升，也带来了精神的愉悦感。
  3. 小伙伴：健身房的环境，刚开始都比较陌生，但时间久了，相互讨论健身方法，就逐渐认识了很多小伙伴，身边有很多努力变好的大家，也带动了自我不断提升。
  4. 健康：健身后自己尿酸过高的毛病也逐渐改善，今年体检已经完全恢复正常了。
  5. 兴趣：当形成了健身的兴趣，逐渐有了深入了解摄影，文学，古典乐等爱好。培养有趣的身体和灵魂，是值得终生投入并叠加性受益的经历。

期望

  1. 找到一个女朋友？心疼自己还单身，感觉还是爸妈催的太紧^_^
  2. 最近不敢增肌？怕是女生都觉得身材太壮不太好，还是对身材进行精雕细琢吧。
  3. 希望能有更多的小伙伴认识并感受到运动的魅力，当然能将自己的身体当作艺术品来打造就更完美了。

最后

分享一段鲁米的，有关自我实现预言的诗句：

注意看那些
在窗户光线里移动的尘屑。
它们的舞蹈就是我们的舞蹈。
我们鲜少用心倾听我们内在的音乐，
但我们莫不随着它起舞。
